# 📦 Inventory Management Dashboard (Microsoft Excel)

## 📌 Project Overview
This project is an end-to-end Inventory Management Dashboard built in Microsoft Excel to streamline inventory operations and provide actionable business insights. It helps monitor products, customers, vendors, purchases, sales, and stock levels through an interactive dashboard.

The dashboard enables users to analyze inventory performance, track stock availability, monitor sales trends, and make informed business decisions using dynamic reports and KPIs.



## 🚀 Features
- Interactive Dashboard with KPI Cards
- Inventory Tracking & Stock Monitoring
- Sales Analysis
- Purchase Analysis
- Customer Management
- Vendor Management
- Product Master
- Pivot Tables & Pivot Charts
- Slicers for Dynamic Filtering
- Data Validation
- Conditional Formatting
